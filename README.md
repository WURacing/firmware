# Code for sensor boards

## [`WUFR23.dbc`](WUFR23.dbc): DBC for custom data system

| CAN ID | Data               |
| ------ | ------------------ |
| 0x1    | RSB accel (x,y,z)  |
| 0x2    | RSB anlg 1 & 2     |
| 0x3    | RSB anlg 3 & 4     |
| 0x4    | RSB anlg 5 & 6     |
| 0x5    | FRSB accel (x,y,z) |
| 0x6    | FRSB anlg 1 & 2    |
| 0x7    | FRSB anlg 3 & 4    |
| 0x8    | FRSB anlg 5 & 6    |
